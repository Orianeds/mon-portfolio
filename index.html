À propos de moi

Je suis

Ces défis n'ont fait que croître ma motivation en confirmant ma conviction que le développement est la voie qui me convient. Je recherche activement une entreprise partageant mes valeurs d'inclusion, de bienveillance et d'ouverture, où je pourrais contribuer avec enthousiasme et des idées innovantes.

Voici mon premier projet personnel, mon portfolio, le site sur lequel vous êtes !

J'ai souhaité faire un portfolio pour montrer les capacités que j'ai acquises depuis le début de ma formation.
J'ai fait le choix de ne pas utiliser de frameworks dans un premier temps pour exploiter au maximum mes connaissances ainsi que pour me challenger.

Je suis fière du rendu de celui-ci et je suis déjà en train de penser aux améliorations que je peux lui apporter et à de nouvelles fonctionnalités :
- ajouter une fonctionnalité pour traduire le site en anglais en appuyant sur un bouton
- ajouter une fonctionnalité pour un dark mode
- ajouter une fonctionnalité pour cacher la barre de navigation latérale et la faire apparaître en glissant la souris dessus

J'ai essayé de faire un code propre et compréhensible pour tous.
L'accessibilité et l'éco-conception sont des notions qui me tiennent à coeur, c'est pourquoi :
- j'ai fait attention aux contrastes
- j'ai utilisé une police lisible
- j'ai converti mes photos en format webp et mes icônes en format svg afin qu'elles soient moins lourdes
- j'ai utilisé l'attribut lazy loading afin que mes photos et vidéos ne se chargent pas automatiquement

Le site est responsive.
N'hésitez pas à me partager vos avis et conseils afin que je puisse m'améliorer.

Jeu des allumettes

Technos utilisées

HTML

CSS

Javascript

Le jeu des allumettes est un petit jeu simple mais amusant pour un duel très stratégique ! Il y a 50 allumettes à retirer, les joueurs tirent chacun leur tour entre 1 et 6 allumettes. Le joueur qui tire la dernière allumette a gagné.

Dans cet exercice, des règles ont été définies pour que :
- l'utilisateur puisse définir un nombre de joueurs
- un message personnalisé s'affiche pour indiquer le joueur qui doit jouer
- les joueurs retirent au moins 1 allumette
- les joueurs ne retirent pas plus de 6 allumettes
- le résultat ne soit pas négatif, les joueurs doivent retirer le nombre exact d'allumettes restant
- une pop-up s'affiche lorsqu'un joueur gagne et précise le nom du joueur gagnant
- après une victoire, la page se recharge pour recommencer une partie

Welcoming page - Extension de navigateur

Technos utilisées

HTML

CSS

Javascript

API

MANIFEST V3

Nombre de participants : 3

Ce projet collectif effectué dans le cadre de ma formation à Ada Tech School nous a permis de créer une extension de navigateur.
Nous avons créé une extension de navigateur (chrome) permettant de personnaliser les nouveaux onglets dès leurs ouvertures pour afficher une page d'accueil.
Les différentes fonctionalités mises en place sont les suivantes :
- Affichage de la date et de l'heure mise à jour automatiquement toutes les secondes.
- Affichage des données météo lié à l'appel d'une API météo et d'une API de géolocalisation.
- Mise à jour du back-ground au rafraîchissement de la page.
- Développement d'une To-Do-List avec sauvegarde des tâches en cours en localstorage.
- Appel API permettant l'affichage des sites les plus consultés par l'utilisateur.
- Développement d'un mini-jeu Mastermind accessible depuis la page d'accueil.

Ce projet a été réalisé en mob-programming

The ALOP's Adventure - Création d'un jeu vidéo

Technos utilisées

Lua

Pico-8

Nombre de participants : 3

Ce projet collectif effectué dans le cadre de ma formation à Ada Tech School a pour objectif de créer un jeu vidéo rétro complet.

Nous avons créé les graphismes, les sons, et le code qui permet de jouer.
Pico-8 est une console imaginaire qui permet de créer, partager, et jouer à des jeux miniatures qui évoquent avec nostalgie les débuts du jeu vidéo grand public.
La console possède un environnement de développement complet, de l'éditeur de code aux graphismes, en passant par la musique.
Ses spécifications sont un écran carré de 128 pixels, 16 couleurs et 4 canaux sonores.

Notre jeu, The ALOP's Adventure, a pour but de sauver notre héros d'un méchant qui l'a kidnappé. Le jeu se déroule en deux temps :
- une première partie sous forme de jeu d'aventure, où nous devons récupérer des clés pour ouvrir les portes pour atteindre Grogu et le libérer de Jabba le Hutt. Une fois que nous l'avons récupérer, s'en suit une course contre la montre pour atteindre notre vaisseau spatial et s'enfuir à temps.
- une deuxième partie sous forme de jeu de shooter, où nous devons abbattre les vaisseaux ennemis et bien évidemment l'ultime vaisseau pour enfin être libre et gagner la partie.

Pour effectuer ces deux jeux nous avons dû gérer les déplacements, les décors, les différentes salles, les vies, les tirs, les collisions, ...

Ce projet a été réalisé en mob-programming.
